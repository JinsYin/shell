# 解压缩工具

| 文件扩展名  | 描述                        |
| ----------- | --------------------------- |
| `*.zip`     | zip                         |
| `*.gz`      | gzip                        |
| `*.bz2`     | bzip2                       |
| `*.xz`      | xz                          |
| `*.Z`       | compress                    |
| `*.tar`     | tar 打包文件，未经压缩      |
| `*.tar.gz`  | tar 打包文件，经 gz 压缩    |
| `*.tar.bz2` | tar 打包文件，经 bzip2 压缩 |
| `*.tar.xz`  | tar 打包文件，经 xz 压缩    |
